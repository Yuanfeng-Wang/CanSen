.. _sec-keywords

=========================
Supported SENKIN Keywords
=========================

The following is a list of the currently supported keywords in the 
SENKIN-format input file::

    CONV, CONP, VPRO, CONT, COTV, VTIM, TTIM, TPRO, TEMP, REAC, PRES,
    TIME, TLIM, DTIGN, ATOL, RTOL, ATLS, RTLS, DELT, DTSV, STPT, EQUI,
    OXID, FUEL, CPROD, ADD, SENS, VOL, END
